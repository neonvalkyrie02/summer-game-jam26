Reactorcore Art (RC Art)
==
PACK: Core Machines

--

Pixel-art sprites pack depicting various machines, reactors and generators, including armor/cover components that go on top of them.

--

All blocks are measured with a 24-pixels per meter/tile standard. An average human would be roughly between 32-40 pixels tall. Suitable for more cartoony proportion characters similar to Super Mario World.

--

By default in image import settings, all sprites/textures have been set to 32 pixels per unit and using a point filter so they appear crisp.
